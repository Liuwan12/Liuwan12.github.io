JS基础学习1

发表于 2017-12-20

1.js基础

  1. js数据类型：基本数据类型boolean，string，number。 空类型null，undefined。es6的symbol.
  2. 复杂数据类型：Array Date Object Regexp Number Boolean String function 首字母大写的是函数
  3. js包含三部分ecmascript(规定js的语法)，bom(浏览器对象)，dom（提供一套完整的页面操作元素的API）

2.检测当前数据类型的方法，也就是typeof关键字

  • typeof 123 //number
  • typeof ‘abc’ // string
  • typeof true //boolean
  • typeof undefined // undefined
  • typeof {} // object
  • typeof [ ] // object
  • typeof console.log() //function
  • typeof null // object

3.异步操作的方法

  • 1.回调函数 eg:setTimeout
  • 2.事件发布订阅
  • 3.promise 将函数调用改为了链式调用
  • 4.generator 消灭了promise的特征方法,需要自己维护yield的执行
  • 5.async/await 优雅的异步操作方法

4.js三种作用域类型（全局，函数，块级）

  1. 全局变量： 声明在函数外部的变量，任何地方都可以访问到该变量
  2. 局部变量： 声明在函数体内，只有函数体内可以访问到该变量
  3. 函数内部声明变量的时候，一定要用var命令，否则，你声明的是全局变量
  4. 在函数外面使用var声明变量后，在函数内部改变该变量的值后，函数外面的该变量的值也随之改变

  5. 在函数外面使用var声明一个变量后，在函数内部使用var再次声明一次并改变其值，函数外面该变量的值不会发生改变

    • 用于修饰常量，定义的变量不可修改，而且必须初始化，声明位置不限

    • let 声明的变量在{}中使用，变量的作用域限制在块级域中

  • js中数组的方法: join(),push(),pop(),shift()–删除数组的第一项，并返回原始数组的值,unshift()–将参数添加到原属组的开头,sort(),reverse(),
    concat(),slice(),splice(),IndexOf(),forEach(),map(),filter(),every()

8.this作用域

  • 每个函数的this是在调用的时候被绑定的，完全取决于调用函数的调用位置而不是声明位置
    1. 默认情况下（非严格模式），没有找到直接调用者，this指向window
    2. 严格模式（’use strict’），没有找到直接调用者，this是undefined
    3. this总是代表它的直接调用者，比如：obj.fun，那么fun中的this是obj
    4. 使用call，apply，bind绑定的this指向的是绑定的对象
      箭头函数中的this指向定义时的this，而不是执行时的this

https://blog.csdn.net/qq_33207292/article/details/78625771 可参考此文

10. 渲染优化

  1. CSS放前面，JS放后面
  2. 懒加载（图片懒加载、下拉加载更多）
  3. 减少DOM查询，对DOM查询做缓存
  4. 减少DOM操作，多个操作尽量合并在一起执行
  5. 事件节流–函数节流所做的工作就是每隔一段时间去执行一次原本需要无时不刻地在执行的函数
